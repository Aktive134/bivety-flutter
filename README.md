# Bivety Bank

## Technology Stack
Node, express, MongoDB, Typescript

 ## Folder organization.
 Domain driven design pattern.

 ## Project description.

An API that allows users to:
- create an account 
- retrieve account details with an account number
- retreive all account numbers
